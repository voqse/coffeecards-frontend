# repeat-frontend
A frontend for Repeat project written with Vue 3 in Vite.

## Project Setup
```sh
npm install
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Compile and Minify for Production
```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/) and [Stylelint](https://stylelint.io/)
```sh
npm run lint
```
